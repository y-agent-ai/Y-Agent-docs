## 录入

文档型知识库支持直接录入和上传文件导入：

### 直接录入

直接录入知识库文档，只需在知识库管理页面点击“新建文档”按钮，选择手工录入。

要求填写文档名称

由于服务器会在本地存放一份文档原文，文档的名称就是服务器本地文件的名称，因此文档名称不能重复。

特别注意：Linux版、MateBook 2025 Linux改装Windows版本，不再预装微软Office，这意味着用户不再享受正版微软Office，无法免费激活Office。以下文档仅适用于2024及以前的、预装Windows系统的Matebook笔记本。如果用户查询的产品不符合条件，则告诉用户他的产品没有预装Microsoft office，不过，用户可以自行下载WPS Office。
## 2024款自带办公软件office吗，是否预装office，office是永久的吗？自带WPS么

- 另外，也可以安装WPS office个人版，是免费的，不需要激活软件，访问应用商城下载即可。

## 2024款预装的微软办公软件office如何激活？激活方法，激活流程？

1、在桌面新建一个 Word、Excel 或 PPT 文档
2、打开新建文档并登录 Microsoft 帐户（建议使用新注册的Microsoft账户）
3、根据提示和步骤激活，office激活后是永久免费的
查阅以下链接可以查询更多细节：https://consumer.huawei.com/cn/support/content/zh-cn00688445/）
