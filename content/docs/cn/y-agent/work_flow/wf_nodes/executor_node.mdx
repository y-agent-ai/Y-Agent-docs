## 什么是执行器节点？
执行器是一个允许用户通过jinja模板语言，来手动出发系统函数的地方。
通过对jinja模板的合理设计，用户可调用相应的系统函数，从而完成预设的固定执行任务。

执行器的工作原理是，直接将jinja模进行渲染，然后解析结果，调用相应的系统函数。
