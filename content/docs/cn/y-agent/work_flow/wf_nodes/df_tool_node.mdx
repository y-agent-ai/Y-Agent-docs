<Accordion title="知识库查询">

知识库查询工具用于查询知识库中的文档。

配置方法：

1. 进入知识库管理页面，点击“工具”标签。
2. 点击“知识库查询”工具。
3. 在“知识库查询”工具页面，选择查询的知识库。
4. 在“查询条件”区域，输入查询的关键字。
5. 点击“查询”按钮，即可查询到符合条件的文档。

</Accordion>

<Accordion title="计算器">

计算器工具用于进行简单计算。

配置方法：

1. 进入知识库管理页面，点击“工具”标签。
2. 点击“计算器”工具。
3. 在“计算器”工具页面，输入计算表达式。
4. 点击“计算”按钮，即可得到计算结果。

</Accordion>
