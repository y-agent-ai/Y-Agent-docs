## 设置 Python

### 安装

1. **下载 Python**: 访问 [python.org](https://www.python.org/downloads/) 并下载适用于你操作系统的最新版本。
2. **安装 Python**: 运行安装程序并确保在点击安装之前勾选“将 Python 添加到 PATH”。

### 选择一个编辑器

您有多种选择:

- **IDLE**: 与 Python 一起捆绑，适合初学者
- **VS Code**: 免费，功能强大的编辑器，具有出色的 Python 支持
- **PyCharm**: 功能齐全的 Python IDE（免费社区版可用）

## Python 基础

### 您的第一个程序

创建一个名为 `hello.py` 的文件并添加此代码:

```python
print("Hello, World!")
```

从命令行运行:

```
python hello.py
```
